# Task Manager App

A responsive task management Single Page Application (SPA) built with React.js using component-based architecture and React Hooks.

---

## 🛠️ Tech Stack

- **Frontend:** React.js, HTML5, CSS3
- **State Management:** React Hooks (useState)
- **Build Tool:** Vite
- **Tools:** Git, GitHub

---

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as complete / incomplete
- 🗑️ Delete tasks
- 🔍 Filter tasks — All, Active, Completed
- 📊 Remaining tasks count
- 📱 Fully responsive design

## ⚙️ Getting Started

### Prerequisites
- Node.js 16+
- npm

### Setup

1. **Clone the repository**
```bash
git clone https://github.com/ellathurai-dev/task-manager-app.git
cd task-manager-app
```

2. **Install dependencies**
```bash
npm install
```

3. **Run the app**
```bash
npm run dev
```

The app will start at `http://localhost:5173`
